## pYin

Yin was a 2002 algorithm for fundamental frequency estimation. It is based on the idea of autocorrelation, which is a mathematical operation that measures the similarity between a signal and a delayed version of itself. 

The pYin algorithm is an improved version of the original Yin algorithm, which is more robust to noise and can handle polyphonic signals. can estimate the voice probability, which is a measure of how likely it is that a given frame of audio contains a voiced sound. 
This can give clearer results and don't under score the user's voice quiality
